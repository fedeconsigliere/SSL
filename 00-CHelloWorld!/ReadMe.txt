TP 0 - Hola Mundo

El compilador a usar es GCC - MinGW-W64 
Versión del compilador 15.1.0 
GCC soporta C23 desde versiones anteriores, por ende es valido para la compilación de la materia. 

Se Utilizó el comando en consola  " gcc HolaMundo.c -o HolaMundo " -> Asi compilamos el ejecutable 

Para ejecutar el programa y probarlo, lo que se hizo en terminal fue ./HolaMundo y ver su devolución (.\HolaMundo en nuestro ejemplo también serviria)

Para ver la versión que tenia instalada de Gcc usé en terminal gcc --version

Para redireccionarlo a un txt, sobre nuestra terminal lo que se hizo fue ejecutar "./HolaMundo > SalidaRedireccionaDelExe"
